Dress Code

Elegante Viñedo, estilo “Italian Toscana Summer”

El evento comenzará al atardecer y será mayormente al aire libre, aunque también hay sectores cubiertos. El dress code es elegante pero les recomendamos venir con zapatos aptos para caminar sobre el pasto.
¡No se suspende por lluvia, tenemos plan B!

Men

Trajes de lino o algodón en colores claros o tonos tierra. Camisas con o sin corbata o con accesorios discretos. No utilizar traje blanco. Traer anteojos y abrigo por si refresca.

Colores y tonos de referencia

Women

Vestidos largos o midi, colores tierra, suaves. Si no tenes estos colores, podés usar negro. Evitar estrictamente el color blanco, nude o tonos similares. Traer anteojos y abrigo por si se refresca.
